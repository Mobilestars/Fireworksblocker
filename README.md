# FireworksBlocker

**Version:** 1.0.0  
**Minecraft Version:** 1.18+  
**Author:** Mobilestars  

## Description
FireworksBlocker is a small Bukkit/Spigot plugin that controls the use of firework rockets in Minecraft.  
It provides two main features:

1. **Block Crossbow + Firework Rocket Usage:** Prevents players from using firework rockets together with a crossbow if disabled.
2. **Block Firework Usage:** Prevents players from using or placing fireworks if disabled.

## Configuration
Settings can be adjusted in `config.yml`:

```yaml
# Enable/disable shooting firework rockets with a crossbow
fireworkCrossbowEnabled: false

# Allow or block placing/using fireworks
fireworkplacementallowed: false
